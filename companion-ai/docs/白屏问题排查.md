# 白屏问题排查指南

## 🔍 第一步：浏览器硬刷新

**最常见原因是浏览器缓存了旧版本代码**

### 操作步骤
1. 打开 http://localhost:5173
2. 按快捷键进行硬刷新：
   - **Windows/Linux**: `Ctrl + Shift + R` 或 `Ctrl + F5`
   - **macOS**: `Cmd + Shift + R`

## 🔍 第二步：检查浏览器控制台

### 打开开发者工具
1. 按 `F12` 或右键点击页面选择"检查"
2. 切换到 **Console（控制台）** 标签
3. 查找红色错误信息

### 常见错误及解决方案

#### 错误1: `Cannot find module`
```
Failed to resolve module specifier "pixi.js"
```
**原因**: npm依赖未正确安装  
**解决**:
```bash
cd frontend_app
rm -rf node_modules package-lock.json
npm install
```

#### 错误2: CORS 跨域错误
```
Access to fetch at 'https://cdn.jsdelivr.net/...' has been blocked by CORS policy
```
**原因**: CDN被网络拦截  
**解决**: 使用最小化测试页面（见第三步）

#### 错误3: `app is undefined` 或 `useChat is not a function`
**原因**: Vue组件编译失败  
**解决**: 检查第四步的编译错误排查

## 🔍 第三步：最小化测试（跳过Vue）

访问纯HTML测试页面（不依赖Vue）：

```
http://localhost:5173/test.html
```

### 预期结果
- 看到"✅ PixiJS加载成功"
- 看到"✅ Live2D库加载成功"
- 看到"✅ 模型加载成功！鼠标移动试试看"
- 页面显示蓝发少女Live2D模型

### 如果test.html也白屏
说明Live2D库本身有问题，检查：
1. 网络能否访问 `cdn.jsdelivr.net`（可能被墙）
2. 浏览器是否支持ES6模块（Chrome 90+, Firefox 88+）

## 🔍 第四步：检查编译错误

### 检查TypeScript编译
```bash
cd frontend_app
npx vue-tsc --noEmit
```

预期输出: `ok`（无任何错误）

### 如果有错误
记录错误信息，检查：
1. 是否是类型错误（可忽略，不影响运行）
2. 是否是语法错误（必须修复）

## 🔍 第五步：检查服务器日志

查看Vite开发服务器是否正常运行：

```bash
# 查看进程
ps aux | grep vite

# 检查端口
netstat -ano | grep 5173
```

### 预期输出
```
TCP    0.0.0.0:5173    ...    LISTENING    [PID]
```

### 如果端口未监听
重启前端服务：
```bash
cd frontend_app
npm run dev
```

## 🔍 第六步：完全重置

如果上述步骤都无效，执行完全重置：

```bash
cd /d/DeskTop/AgentGril/companion-ai/frontend_app

# 1. 停止所有node进程
taskkill /F /IM node.exe  # Windows
# pkill node              # Linux/macOS

# 2. 清理依赖和缓存
rm -rf node_modules package-lock.json .vite dist

# 3. 重新安装
npm install

# 4. 重新启动
npm run dev
```
